Query Parameters

Parameter Type Required Description

pageSize Integer Optional Number of items per page (default: 10)

search String Optional Search products by name or description

POST Create Product

Create a new product in the system.

Request Body

Field Type Required Description
name String Required Product name (cannot be empty)
imageUrl String Required URL to product image
rating Float Required Product rating (0-5)
price Double Required Product price (must be > 0)
description String Optional Product description (default: "")
category String Optional Product category (default: "General")
stock Integer Optional Available stock (default: 100)

Test Request

Response Example
